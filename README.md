Multipurpose Flight Computer
============================
This project is a flight computer designed to be used for many purposes, whether it be RC airplanes, model rockets, or even drones. Because of the broad usage capability it's designed to be as small as possible. It includes a barometer, IMU, servo outputs, and a battery controller. It is all powered by the STM32

Why this?
=========
Well there I was... sitting bored at home

Just kidding. I previously made an Arduino Nano based flight computer with premade components and it was my first ever PCB! As I've learned more though, I wanted to challenge myself with something a little more complex and this was the perfect idea! I followed @NotARoomba's tutorial, but tried to make my version as small and compact as possible.
